Google Snippet Maker

Choose a txt-file that contains snippet content.

Per snippet, following information is required divided by a new line: id, url, title, text

The txt-file can have multiple snippet informations. If no information for a field exists, leave an empty line.

Google Snippet Maker can shorten fields, step throug the snippets, download current snippet as jpeg or all snippet as zipped jpegs.
